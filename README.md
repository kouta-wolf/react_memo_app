# シンプルメモアプリ

## 概要
メモアプリの基本的なCRUD動作をReactで実践する（非保存）

## リンク
完成次第

## 学習の目的
- Reactの `useState`で配列・オブジェクトのイミュータブルの扱いを学ぶ
- バックエンドに繋げられるようなデータの扱い方を学ぶ

## 苦労した部分
- 状態管理がどんどん複雑になり、とにかく一覧で見れるということを考えて全て１つのファイルで実装した
- ()や{}の閉じ忘れがどんどん酷くなり修正にかなりの時間がかかった

## 今後もし改善するなら
- TailWindCSSにてUI/UXを調整してモダンなアプリにする
- バックエンドを入れて非保存から保存状態に切り替える

## 最終出力
- GitHubPagesへ出力

## 使用技術
- React (Vite)
- Tailwind CSS
